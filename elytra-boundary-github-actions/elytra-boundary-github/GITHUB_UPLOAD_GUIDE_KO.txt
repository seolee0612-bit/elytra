1) github.com에서 New repository 생성
2) 저장소에서 Add file > Upload files 선택
3) 이 폴더 안의 모든 항목을 업로드
   중요: elytra-boundary-github 폴더 자체가 아니라 그 안의 .github, src, build.gradle 등을 업로드
4) Commit changes
5) Actions 탭 > Build Fabric Mod
6) 초록색 체크가 뜬 실행 클릭
7) 페이지 아래 Artifacts의 elytra-boundary-fabric-1.21.1 다운로드
8) ZIP 안의 elytra-boundary-1.0.0.jar를 서버 mods 폴더에 업로드
9) 서버 완전 재시작
